# 3. 머지 커밋 skip

- Status: Accepted
- Date: 2026-06-08
- Source: PRD (#1), Q5

## Context

머지 커밋은 다른 브랜치가 가져온 줄을 자기 diff에 다시 포함할 수 있다. 그대로 세면 원본
커밋과 머지 커밋에서 같은 줄이 두 번 카운트된다.

## Decision

- 부모가 **2개 이상**인 커밋(머지 커밋)은 numstat·commits 양쪽에서 **완전히 제외**한다.
- `git log --no-merges` 관례와 일치.

## Consequences

- 머지가 가져온 줄이 원본 커밋과 함께 이중 카운트되지 않는다.
- **수용된 한계**: 머지 충돌 해결로 머지 커밋에만 손으로 친 줄은 어디에도 잡히지 않는다.
- 향후 `--first-parent` 같은 대체 머지 모드는 별도 결정 사항(현재 범위 밖).
